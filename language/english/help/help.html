Help: Spotlight

DESCRIPTION


This module puts a News item in a central spotlight block. It has the following features:

  • Create a block displaying the intro text of a given news article
  • Add a custom image to the spotlighted article block
  • Control look and feel through block template
  • Display other recent news below spotlighted item
  • Admin lenth ot titles through block admin
  • Admin order and number of other recent news items displayed.
